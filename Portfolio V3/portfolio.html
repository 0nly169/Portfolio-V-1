Personal project - Blog

This is my personal blog that I build. I used Bootsrap v4 for the main base of my blog then used custom CSS to fully customize it to my liking. I am a fan of mobile first so this project was built with that in mind so it is fully responsive using bootsraps flexible grid system and by writing my own media queries. You can see a large image as the header, this at interval of 10 seconds transitions into another image. I required the use of Javascript to make this possible.
The scource files can be found at Github here.

Coding skills used:

  • HTML
  • Flexbox
  • Bootsrap
  • Javascript

The main issue I faced while making my blog was the responsive aspect. On smaller screens I wanted to have the lastest blog post show up at the start of the page and not have my 'about' section in the way. I used Bootstraps' sidebar-module class to accomplish this.

My Website

My orignal plan for my site was to build a static single page website as you can see in the video. I was happy with it but felt I could do more in terms of customization. With the added content I had to add multiple meda queries to keep my site mobile responive.

I slightly underestimated the work load difference between a single page site and a multiple page website. I found it to be a great experience because it puts more emphasis on clean and readable code. Things can become confusing fast if you dont use comments throught your code.

As you can see in the images my site is mobile responsive. The nav bar collaspes when a specific break-point is reached. All images and other elements are responsive leading to a clean simple to use single column layout on mobile devices and smaller screens. I feel this gives the site a professional finish. Why not resize your browser window to see the changes for yourself?.

On the contact page you can see the inclusion of custom icons, I used the font awesome library to for that.
